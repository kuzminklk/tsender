
### Source
Study project from Cyfrin Updraft Full-Stack Web-3 course

### Description
Web UI for interaction with TSender smart contract for sending a lot of tokens, like for airdrop.

### Technologies
TypeScript, Next.js (React Framework), Rainbow Kit (For Wallet connection), Wagmi (For interactions with smart contracts), React-Query, Vitest (Testing Framework)

### Development Path
1. Develop core functionality and UI during course (1–13 lessons)

### Additional
«tsender-deployed.json» — Anvil blockchain state with predeployed TSender smart-contract
